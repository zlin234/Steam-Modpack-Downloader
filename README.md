Here is a simple guide on how to use it.

1. Download everything and extract somewhere
2. Install python if you do not already have it
3. Get steamCMD and configure the path in the python file if needed
4. Get a list of all the ids in the modpack (refer to the second guide if you don't know how to do this) and put it in a new text file called mods.txt in the same folder/dir as the python file.
5. Please remember to seperate each mod id with a line break
6. Run the python file and wait
7. Navigate to the download section of steamCMD and move the batch file to it (would be something like C:\SteamCMD\steamapps\workshop\content\233860 if you were following along with the default path)
8. Run the batch file and it should work

Let me know on reddit post if it does not work as intended

DISCLAIMER/WARNING: This has not been thoroughly tested and it is recommended to backup whatever you do not want to lose in the downloads section

Guide on how to get the ids of a collection (steam)
Go to the steam workshop url (should look somewhat like this https://steamcommunity.com/sharedfiles/filedetails/?id=2967435666)
Copy the number at the end (in this case, 2967435666)
Put the collection ID at this website https://getcollectionids.moonguy.me/
Copy the entire text and paste into https://www.i2text.com/add-line-breaks and set the setting to ;
Copy the output and paste into the mods.txt and remember to save it
